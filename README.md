# Design and Implementation of Machine Learning Model for Predicting India's GDP

## Overview
The GDP Prediction and Forecasting Dashboard is an end-to-end Machine Learning and Time Series Analytics project developed to analyze historical economic data, predict Gross Domestic Product (GDP), and forecast future GDP trends. The project combines traditional Machine Learning models with Deep Learning and Time Series Forecasting techniques to provide economic insights through an interactive Streamlit dashboard.

## Features
GDP prediction using:
Linear Regression 
Random Forest Regressor
XGBoost Regressor
GDP forecasting using:
LSTM (Long Short-Term Memory)
ARIMA (AutoRegressive Integrated Moving Average)
Interactive Streamlit dashboard for data exploration and model predictions
Economic trend analysis and visualization
Correlation heatmaps for feature relationship analysis
Model comparison using R² Score, RMSE, and MAE
Future GDP forecasting with confidence intervals
Recommendations and performance insights dashboard

## Technologies used
Python
Streamlit
Pandas
NumPy
Scikit-learn
XGBoost
TensorFlow / Keras
Statsmodels (ARIMA)
Plotly

## Project Workflow
1. Data Collection and Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Engineering and Correlation Analysis
4. Machine Learning Model Training and Evaluation
5. GDP Forecasting using LSTM and ARIMA
6. Interactive Dashboard Development with Streamlit
7. Model Comparison and Performance Evaluation

## Evaluation Metrics
The models are evaluated using:
R² Score
Root Mean Squared Error (RMSE)
Mean Absolute Error (MAE)

## Results
The project successfully predicts GDP using Machine Learning models and forecasts future GDP trends using LSTM and ARIMA. Interactive visualizations and model comparison dashboards help analyze economic indicators, identify patterns and support data-driven decision-making.
